HTML
    index : 심플하고 구분이 쉽도록 좌, 우로 공간을 나누어 페이지를 작성했습니다.
    sub page : 상단 네비게이션을 채택하고 , 설명을 위해 상, 하단으로 나누어 컨텐츠를 주었습니다.

CSS 
    index : 좌우 공간을 50%씩 나누었고, sub section 부분은 마우스 호버시 애니메이션을 주었습니다.
    sub page : 상단 네비게이션에 투명도 변화 애니메이션을 채택했습니다.

Javascript
    index : section box 전체를 링크화 하기 위해 자바스크립트로 링크를 걸어주었고, 추가적으로 기능 하나를 더 사용하였습니다.
    sub page : 페이지를 클릭하면 소리가 나는 자바스크립트를 사용하였습니다.

그 외 : index 페이지에선 애니메이션을, sub page 에선 멀티미디어 자료들(텍스트, 사진, 동영상, 사운드, 애니메이션)을 모두 사용하여 구성했습니다.
